專案概述

本專案為一套從硬體感測、即時通訊、桌面應用到 Web 資料查詢的完整系統實作。

系統可即時接收多項生理感測資料，並同步顯示於桌面應用程式，同時將資料儲存至資料庫，提供 Web 介面進行歷史資料查詢與視覺化分析。

功能說明
即時生理監測

SpO₂（血氧濃度）

Heart Rate（心率）

ECG（心電訊號）

溫度 / 濕度

桌面應用程式（WinForms）

即時生理數據顯示

感測模組啟動 / 停止控制

COM Port / 藍牙通訊設定

多模組資料切換（SpO₂ / ECG / HR）

Web 健康量測系統

使用者登入機制

歷史量測資料查詢

圖表化顯示生理數據趨勢

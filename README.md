<h1>專案介紹</h1>

為一套從硬體感測、即時通訊、桌面應用到 Web 資料查詢的完整系統實作。
系統可即時接收多項生理感測資料，並同步顯示於桌面應用程式，同時將資料儲存至資料庫，提供 Web 介面進行歷史資料查詢與視覺化分析。

<h1>功能說明</h1>
<h2>即時生理監測</h2>
<ul>
  <li>SpO₂（血氧濃度）</li>
  <li>Heart Rate（心率）</li>
  <li>ECG（心電圖）</li>
  <li>溫溼度</li>
</ul>  
<h2>桌面應用程式（WinForms）</h2>
<ul>
  <li>即時生理數據顯示</li>
  <li>整合多種感測模組</li>
  <li>COM Port / Bluetooth 連線</li>
  <li>溫溼度</li>
</ul>
